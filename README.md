## Features
- Connect Memory ID or wallet
- Fetch and display data graph

## Setup

```bash
git clone https://github.com/Boehner/memboard.git
cd memboard
npm install
add .env and VITE_MEMORY_API_KEY=mem_yourkeyhere
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).
